# 🏦 Loan Approval Prediction using Logistic Regression

## 📌 Project Overview

This project predicts whether a loan application will be approved or rejected using the Logistic Regression algorithm from Scikit-learn. The model is trained on a sample dataset containing applicants' income and credit scores. After training, the program accepts income and credit score as user input and predicts whether the loan will be approved.

---

## 🚀 Features

- Created a sample dataset containing applicants' income and credit scores.
- Trained a Logistic Regression model using Scikit-learn.
- Takes income and credit score as input from the user.
- Predicts whether the loan will be approved or rejected.
- Displays the prediction in a simple and user-friendly format.

## 📖 How the Program Works

The program first creates a small dataset containing applicants' income and credit scores along with their loan approval status. It then trains a Logistic Regression model using this data. After training, the user enters a new applicant's income and credit score. The model analyzes these values and predicts whether the loan should be approved or rejected based on the patterns it learned from the training data.
